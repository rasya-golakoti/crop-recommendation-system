# 🌱 Smart Crop Recommendation System

## Overview

This project uses Machine Learning to recommend the most suitable crop based on soil nutrients and environmental conditions.

The system predicts crops using:

* Nitrogen (N)
* Phosphorus (P)
* Potassium (K)
* Temperature
* Humidity
* pH
* Rainfall

## Dataset

Crop Recommendation Dataset

* 2200 samples
* 22 crop classes
* Balanced dataset

## Machine Learning Workflow

1. Data Understanding
2. Exploratory Data Analysis (EDA)
3. Data Preprocessing
4. Label Encoding
5. Train-Test Split
6. Model Training
7. Model Evaluation
8. Model Deployment

## Models Used

### Decision Tree

Accuracy: 97.95%

### Random Forest

Accuracy: 99.55%

### XGBoost

Accuracy: 98.86%

## Best Model

Random Forest achieved the highest accuracy and was selected for deployment.
